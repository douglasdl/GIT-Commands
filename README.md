# GIT-Commands

## Commands

Check a directory settings
<br>
<code>
    git config --list
</code>
<br>

Set or change the user name
<br>
<code>
    git config --global user.name "Douglas Dias Leal"
</code>
<br>

Set or change the user e-mail
<br>
<code>
    git config --global user.email "douglas_san@hotmail.com"
</code>
